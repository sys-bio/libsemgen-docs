Biomodels Qualifiers¶

class bqb¶

This class contains all members from the collection of BioModels biolocial qualifers. BioModels biology qualifiers are used as predicates in RDF to annotation singular and composite annotations. The most important relation is bqb::is, which is used to define elements in a model. When building annotations, you should always define elements with a single definition URI using SingularAnnotation::addDefinition or Entity::addDefinition on the Entity object contained in a CompositeAnnotation (which you can get using CompositeAnnotation::getEntity). When you define elements using one of these functions, the definitions will automatically use the bqb::is qualifier.

Another important qualifier is bqb::isPropertyOf. This is automatically inserted by libSemSim when serializing the PhysicalProperty of CompositeAnnotations.

The members are:

Relation qual = bqb::hasProperty;
